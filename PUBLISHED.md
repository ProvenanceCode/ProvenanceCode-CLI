## Installation Methods

Users can install and use the CLI in three ways:

### 1. npx (No Installation - Recommended)

```bash
npx provenancecode-cli install
npx provenancecode-cli config set --app-code=MYAPP
npx provenancecode-cli validate
```

### 2. Global Installation

```bash
npm install -g provenancecode-cli
prvc install
prvc validate
```

### 3. Project Dependency

```bash
npm install --save-dev provenancecode-cli
npx prvc install
```
